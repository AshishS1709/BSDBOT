## How the Matching Works

The chatbot uses **keyword matching** to find the best FAQ response. When a user asks a question, it:

1. Converts the message to lowercase
2. Checks for keyword matches in each FAQ
3. Scores each match (longer phrases = higher score)
4. Returns the best match with confidence score
5. Falls back to general response if confidence is low
